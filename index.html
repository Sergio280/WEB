Casos de Uso Práctico

Flujos de trabajo documentados con resultados verificables para proyectos de ingeniería estructural

Generación de Planilla de Despiece de Armadura (BBS) — Del modelo Revit al archivo Excel

01
Modelar barras de refuerzo en Revit

El proyectista coloca elementos Rebar con el tipo de forma (shape code), diámetro nominal y parámetros de gancho correctamente definidos según el sistema de barras del proyecto.

02
Ejecutar Set Marks

BIMS agrupa las barras por geometría, diámetro y longitud mediante un algoritmo de hash geométrico. Asigna marcas secuenciales y escribe automáticamente los parámetros compartidos: BIMS_A…BIMS_L, BIMS_ShapeCode, BIMS_Length, BIMS_HookStart, BIMS_HookEnd y más.

⏱ ~30 s para 500 barras
03
Ejecutar Shape Images

Genera una imagen PNG por cada marca única con cotas nominales y longitudes de gancho sobreimpresas. Las imágenes se almacenan localmente para incrustarlas automáticamente en la planilla BBS.

⏱ ~2 min para 80 marcas distintas
04
Ejecutar Bar Bending Schedule (BBS)

Crea una Schedule View en Revit con las columnas: Marca · Tipo · Ø · Cantidad · Longitud total · Dimensiones (A, B, C…) · Ganchos (inicio/fin) · Imagen de forma.

05
Exportar a Excel o BVBS

Exportar BBS Excel produce un .xlsx con formato profesional, listo para entregar a obra y contratistas. Exportar BVBS genera el formato estándar para máquinas de doblado CNC automatizadas.

⏱ Proceso completo: 45 min vs. 8 h en proceso manual

Encofrado Automatizado — Columnas, Vigas, Losas y Escaleras en un solo comando

01
Seleccionar elementos estructurales

Seleccione columnas (Structural Column), vigas (Beam), losas (Floor) o escaleras (Stair) en el modelo. BIMS identifica automáticamente la categoría de cada elemento seleccionado.

02
Ejecutar Encofrado

El algoritmo clasifica cada elemento y genera los componentes correspondientes:

Columnas → Muros de encofrado perimetrales
Vigas → Muros laterales + Suelo de fondo
Losas → Suelo de encofrado horizontal
Escaleras → Muros verticales + Suelos inclinados

⏱ ~20 min vs. 4 h en proceso manual
03
Recortes automáticos por geometría adyacente

BIMS detecta la geometría de los elementos estructurales contiguos y aplica recortes (Void Geometry) para evitar solapos entre encofrados de vigas y columnas, preservando curvas y geometría compleja.

04
Resultado: elementos nativos de Revit

El encofrado generado son elementos Wall y Floor nativos, cuantificables y detallables directamente en Revit sin procesamiento adicional. El tipo de muro es configurable en el diálogo de Encofrado.

Acero de Refuerzo en Columna Rectangular — Barras Longitudinales y Estribos con Zonas de Confinamiento

01
Seleccionar columnas estructurales

Seleccione una o múltiples columnas de categoría Structural Column. BIMS lee automáticamente la sección transversal, la altura libre y el recubrimiento nominal configurado.

02
Configurar y ejecutar Acero Columnas

Defina en el cuadro de diálogo: diámetro de barra longitudinal, número de barras por cara, recubrimiento de diseño y tipo de sección (rectangular o circular). BIMS coloca las barras con posicionamiento exacto respecto a la geometría de la columna.

⏱ ~15 s por columna
03
Ejecutar Estribos Columnas

BIMS calcula y genera las zonas de confinamiento según normativa sismorresistente: la longitud de confinamiento corresponde a max(hcol, llibre/6, 500 mm). Los estribos se generan con ganchos a 135°.

Zona confinada: Ø8 @100 mm  |  Zona central: Ø8 @200 mm

04
Resultado integrado con el flujo BBS

Todas las barras generadas son elementos Rebar nativos de Revit. Se integran directamente con el flujo Set Marks → Shape Images → BBS → Exportar Excel para documentación completa sin trabajo manual adicional.

Importación DWG (SolidWorks/AutoCAD) → DirectShape en Revit → Exportación NWC para Navisworks

01
Recibir archivo DWG con geometría 3D

El proveedor o fabricante entrega un archivo .dwg con bloques 3D (equipos MEP, perfiles metálicos, prefabricados, maquinaria) exportados desde SolidWorks, Inventor u otro CAD mecánico.

02
Ejecutar Importar DWG

BIMS convierte cada bloque DWG en un DirectShape de Revit con la geometría sólida exacta. Los elementos conservan posición, escala y orientación del archivo original. Opción Importar DWG Independiente para DirectShapes no vinculados a un nivel.

⏱ ~2 min para 50 bloques complejos
03
Ejecutar Exportar NWC

Reasigna los DirectShapes a categorías de Revit compatibles con Navisworks: Duct, Pipe, MechanicalEquipment, StructuralFraming, etc. El modelo queda listo para coordinación BIM en Navisworks Manage o Clash Detective.

04
Calcular Volúmenes (opcional)

Analiza y escribe el volumen real de cada DirectShape en un parámetro de instancia, útil para cuantificación de materiales, estimación de peso y presupuestos de obra.

  • Vigas → Muros laterales + Suelos inferiores

Marcas Pulldown

Gestión completa de la numeración de barras

  • Set Marks — Agrupa por forma/diámetro/longitud y asigna marcas secuenciales
  • Edit Mark / Renumber — Edición y cambio de número de marca
  • Match Rebar — Copia propiedades entre barras de refuerzo

Etiquetado automático de barras en vistas

  • Bar Bending Schedule (BBS) — Planilla Revit: Marca | Tipo | Ø | Cantidad | Longitud | Dimensiones | Ganchos | Imagen

Verificar + Config Pulldown

Control de calidad y personalización

  • Browse — Navegador de marcas con búsqueda en tiempo real

  • Update — Refresca todos los parámetros BIMS sin reasignar marcas

Parámetros Compartidos BIMS

Escritos automáticamente por Set Marks en cada barra

  • BIMS_A … BIMS_L — Dimensiones de la barra (redondeadas)
  • BIMS_DimensionsText — Texto pre-formateado: "A= 1380; B= 480; C= 1160"
  • BIMS_HookStart / End — Longitud de gancho inicio/fin
  • BIMS_HookLengthsText — Texto pre-formateado: "S= 145; E= 145"
  • BIMS_Length / Diameter — Longitud total y diámetro nominal
  • BIMS_ShapeCode — Código de forma (BS 8666 / DIN 1356)
  • BIMS_Member / Layer — Elemento estructural y capa de refuerzo
  • BIMS_Assignment / Revision — Plano asignado y revisión

Preguntas Frecuentes

Todo lo que necesitas saber antes de empezar

¿Qué versiones de Revit son compatibles? ▾
BIMS es compatible con Revit 2022, 2023, 2024, 2025 y 2026. Las versiones 2025 y 2026 usan .NET 8.0; las versiones 2022–2024 usan .NET Framework 4.8. Ambas se instalan automáticamente desde el Autodesk App Store.
¿Necesito tarjeta de crédito para el trial? ▾
No. El trial de 30 días se activa únicamente con tu dirección de correo electrónico. No se solicita ningún dato de pago hasta que decidas contratar un plan.
¿Qué pasa cuando vence el período de prueba? ▾
Al vencer los 30 días, BIMS se desactiva automáticamente. Tus proyectos y datos en Revit no se ven afectados en absoluto. Para continuar, elige un plan y recibirás tu clave de activación por email.
¿Puedo instalar BIMS en más de un equipo? ▾
El Plan Individual permite la instalación en 1 equipo. El Plan Profesional permite hasta 3 equipos para el mismo usuario. Para equipos más grandes, el Plan Empresa no tiene límite de seats — contacta ventas para un presupuesto.
¿Cómo se instala y actualiza BIMS? ▾
BIMS se descarga e instala desde el Autodesk App Store con un solo clic. Las actualizaciones se notifican y gestionan automáticamente desde el mismo App Store, por lo que siempre tendrás la última versión disponible.
¿BIMS funciona con modelos enlazados (Revit Links)? ▾
Sí. Varios comandos —en especial del módulo Encofrado y Tarrajeo por Habitación— están diseñados para trabajar con modelos vinculados, reconociendo muros, columnas y otros elementos de documentos enlazados con transformaciones correctas.
¿Qué soporte incluye cada plan? ▾
El Plan Individual incluye soporte por email con respuesta en 48 h. El Plan Profesional incluye soporte prioritario (respuesta en 24 h) y acceso anticipado a funciones beta. El Plan Empresa incluye soporte dedicado, capacitación y personalización bajo pedido.
